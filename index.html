Projet Principal

🎯 Pipeline d'Analyse de Satisfaction Client

Showroom Privé - Projet de formation DataScientest

⭐ PROJET PHARE

Objectif : Analyser, modéliser et visualiser la satisfaction client à partir de plus de 20 000 avis réels collectés sur Showroom Privé via un pipeline complet de bout en bout.

🏗️ Architecture complète

Pipeline end-to-end : MongoDB → Airflow → MySQL → FastAPI → Prometheus/Grafana

  • Ingestion automatisée : Migration des données brutes depuis MongoDB vers MySQL avec orchestration Airflow
  • Modélisation relationnelle : Schéma SQL optimisé avec scripts de transformation et tests de qualité des données
  • Machine Learning : Pipeline d'analyse de sentiments (TF-IDF + Régression Logistique, RandomForest, SVM, CamemBERT) avec réentraînement automatique
  • API REST production-ready : FastAPI avec documentation Swagger (endpoints /predict, /stats, health check)
  • Containerisation complète : Architecture Docker multi-services (MySQL, MongoDB, Airflow, API, monitoring)
  • Observabilité : Stack complète Prometheus + Grafana + StatsD pour métriques temps réel

🔧 Stack technique & DevOps

  • Data Engineering : Pipeline ETL modulaire avec code Python structuré (src/ingestion, src/transform, src/utils)
  • Orchestration : DAGs Airflow pour automatisation complète des traitements
  • Qualité du code : Black (formatage), Flake8 (linting), Pytest (tests unitaires)
  • CI/CD : GitHub Actions (build, tests, staging automatisés)
  • Sauvegarde & Restauration : Scripts automatisés MySQL (mysqldump) et MongoDB (mongodump)
  • Documentation : README technique complet, ERD, architecture diagrams, OpenAPI/Swagger

📊 Résultats & Impact

  • ✅ 20 000+ avis clients traités et analysés automatiquement
  • ✅ Réduction de 75% du temps de traitement manuel grâce à l'automatisation
  • ✅ Pipeline production-ready : reproductible, testé, documenté et monitoré
  • ✅ API REST déployée avec prédictions ML accessibles via endpoints REST
  • ✅ Infrastructure scalable containerisée avec Docker Compose
  • ✅ Monitoring temps réel avec dashboards Grafana et alerting Prometheus
